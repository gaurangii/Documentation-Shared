 Convertalk gives you the power to validate the email entered in the chat. You can enable it in the edit your chat flow options by clicking on the message which is asking for E-mails.

You can set it up to either check for the basic format i.e., [abc@example.com](mailto:abc@example.com), or for a particular domain. For example, if you only want customers to come from  Convertalk, you can add a filter setting as include specific domains, [ Convertalk.com](http:// Convertalk.com).
